# 同步删除助手

同步删除助手是一个 MoviePilot V2 插件，用于在 MoviePilot 删除媒体相关记录或文件时，同步清理 qBittorrent、Transmission 下载任务，并按配置处理硬链接媒体文件。

## 默认策略

- 插件默认关闭，需要手动启用。
- 删除原始下载数据默认开启。
- 硬链接清理范围默认是仅当前文件。
- 下载器删除失败时默认不继续删除硬链接。
- 实际删除必须命中媒体目录或下载目录白名单。
- 仅标题匹配不会自动删除，会进入待确认记录。

## 配置说明

- 启用插件：打开后开始监听删除事件。
- 演练模式：只记录将要执行的动作，不实际删除。
- 删除原始下载数据：删除下载器任务时同时删除下载器侧数据文件。
- 手动执行目标路径：填写一个下载源文件路径，或填写已经入库的媒体硬链接路径；插件会按硬链接反查下载器任务。
- 立即执行一次：保存配置后立刻按手动执行目标路径运行一次；开启演练模式时只记录预演结果。
- 插件日志：立即执行会输出开始和完成日志，完成日志会包含状态、原因、下载器、任务和路径。
- 硬链接清理范围：选择仅当前文件或同任务全部媒体硬链接。
- 媒体目录白名单：允许删除媒体硬链接的目录，可从候选目录选择，也可手动输入。
- 下载目录白名单：允许匹配和删除下载源数据的目录，可从候选目录选择，也可手动输入。
- 路径扫描根目录：插件会从这些根目录扫描候选目录，默认包含 `/vol2/1000/media`、`/media`、`/downloads`、`/mnt`、`/data`、`/volume1`。
- 下载器删除失败仍继续清理硬链接：高级选项，默认关闭。

## 建议验证流程

1. 启用插件并打开演练模式。
2. 在媒体目录白名单和下载目录白名单里选择候选路径；如果没有候选项，就手动输入 MoviePilot 容器内路径，例如 `/vol2/1000/media/movie`、`/vol2/1000/media/tv`、`/vol2/1000/media/download`。
3. 如需手动测试，填写手动执行目标路径，例如 `/vol2/1000/media/download/xxx.mkv`，也可以填 `/vol2/1000/media/movie/xxx.mkv`、`/vol2/1000/media/cartoon/xxx.mp4` 这类媒体硬链接路径，然后打开立即执行一次并保存。
4. 打开插件日志，确认出现 `立即执行开始` 和 `立即执行完成`。如果状态是 `dry_run` 表示演练成功，`skipped` 表示没有匹配到下载任务，`failed` 表示被白名单或下载器删除结果拦截。
5. 或删除一个测试媒体文件或历史记录。
6. 在插件详情页确认匹配到的下载器任务和计划删除路径。
7. 关闭演练模式，用测试下载任务验证 qBittorrent 或 Transmission 删除行为。
8. 验证下载器删除失败时媒体硬链接不会被继续删除。

## 安全边界

插件不会全盘扫描硬链接，不会根据标题直接自动删除任务，也不会删除白名单外路径。
